🏫 Hora certa - Extensão Chrome

Hora certa é uma extensão simples para Chrome desenvolvida como projeto do Bootcamp II. Ela foi construída usando Manifest V3 e tem como objetivo facilitar o dia a dia do aluno do Bootcamp.

🚀 Funcionalidades

Interface simples e intuitiva

Acesso rápido às principais ferramentas do Bootcamp

Desenvolvida como projeto de aprendizado de Chrome Extensions

💻 Instalação

Baixe o arquivo .zip na página de Releases
.

Descompacte o arquivo em uma pasta de sua preferência.

No Chrome, acesse chrome://extensions/ e ative o Modo de Desenvolvedor.

Clique em Carregar sem compactação e selecione a pasta descompactada.

Pronto! A extensão estará instalada e pronta para uso.
