# Minha Extensão do Bootcamp

Extensão simples para Google Chrome usando Manifest V3.

## Como instalar

1. Clone ou baixe este repositório.
2. Acesse `chrome://extensions` no Chrome.
3. Ative o modo desenvolvedor.
4. Clique em "Carregar sem compactação".
5. Selecione a pasta do projeto.

## Estrutura

- `manifest.json`: arquivo de configuração da extensão.
- `src/`: código fonte da extensão.
- `icons/`: ícones da extensão.
- `docs/`: página web para GitHub Pages.
